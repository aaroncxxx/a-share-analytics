## ✨ Features / 功能

- 🔍 **微博热搜抓取** — 实时抓取 A 股相关热搜关键词
- 📈 **大盘指数** — 上证/深证/创业板实时行情
- 🟢 **涨停板/跌停板** — TOP 列表 + 涨停原因分析
- 📊 **热门板块** — 行业板块涨跌排行
- 🔗 **关联分析** — 热搜 vs 行情交叉分析
- 📱 **多种输出** — 完整报告 / 精简模式 / JSON

## 🚀 Quick Start / 快速开始

```bash
pip install akshare
python scripts/analyzer.py
python scripts/analyzer.py --brief
python scripts/analyzer.py --json
```
