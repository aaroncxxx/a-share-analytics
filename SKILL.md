## Usage

```bash
python3 "{baseDir}/scripts/analyzer.py" [options]
```

### Options

| Flag | Description |
|------|-------------|
| `--json` | JSON 格式输出 |
| `--brief` | 精简模式：仅热搜 + 涨停 |
| `--trend` | 趋势模式：含大盘5日走势 |
| `--md` | Markdown 格式输出（方便分享到群/文档） |
| `--no-weibo` | 跳过微博数据（只看行情） |
| `--no-market` | 跳过行情数据（只看热搜） |
| `--stock <代码>` | 查询单只股票详情 |
| `--watchlist <股票1,股票2>` | 自选股过滤（逗号分隔） |
| `--guba` | 东方财富股吧热帖 |
| `--news` | 财经新闻聚合 |
| `--all` | 全部数据（股吧 + 新闻） |
| `--xueqiu` | 雪球讨论（需配置 token） |
| `--version` | 显示版本号 |
| `-h, --help` | 显示帮助信息 |

## 依赖

- Python 3.8+
- akshare (`pip3 install akshare`)

## 自定义配置

- 修改 `SECTOR_KEYWORDS` 列表可以添加你关注的板块关键词
- 修改 `EXACT_KEYWORDS` 列表可以调整精确匹配规则
- 快照缓存在 `scripts/.cache/` 目录，可手动删除历史数据

## Troubleshooting

| 问题 | 解决方案 |
|------|---------|
| `ModuleNotFoundError: No module named 'akshare'` | 运行 `pip3 install akshare` |
| 微博热搜获取失败 | 检查网络，微博 API 有时限流，稍后重试 |
| 涨停数据为空 | 非交易日/盘前数据为空是正常的 |
| 个股查询未找到 | 确认股票代码是6位数字（如 688256） |
| 北向资金数据为0 | 非交易时段北向资金为0是正常的 |
